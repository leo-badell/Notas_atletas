A seguinte é a primeira entrega dos projetos de certificação do curso DEVstart que está sendo promovido pelo Floripa mais tec. 
O trabalho a seguir, “Pontuação dos atletas”, mostra uma matriz de objetos com as notas de vários atletas. 
Portanto, utilizei o método forEach junto com os métodos slice() e sort() para poder iterar nas notas. 
Também utilizei o método reduce() para poder calcular a soma das notas intermediárias. 
Ao final, fiz o cálculo da média com uma variável e a imprimi no console utilizando console.log().
